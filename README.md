# Aplicación de Visualización de Consultas - Atenea

Esta aplicación es una interfaz frontend pensada para interactuar con la API **Atenea**. Su objetivo principal es ofrecer una forma fácil y visual de consultar datos sobre personas y sus registros en diferentes áreas, como **pagos**, **salud** e **imponibles**.

## Funcionalidades

-   **Consultas sobre Personas**: Permite buscar y obtener información detallada sobre personas utilizando su identificador o datos personales.
-   **Pagos**: Muestra los registros de pagos asociados a cada persona, con detalles sobre transacciones, montos, fechas y estados.
-   **Salud**:
    -   **Emergencias**: Accede a los registros de emergencias médicas, incluyendo detalles sobre eventos, fechas y respuestas.
    -   **Turnos**: Consulta los turnos médicos programados, su estado y los profesionales a cargo.
-   **Imponibles**:
    -   **Rodados**: Visualiza la información relacionada con vehículos, como los registros y pagos asociados.
    -   **Inmuebles**: Consulta datos sobre inmuebles, incluyendo su ubicación, valor y pagos de impuestos.

## Características

-   **Interfaz intuitiva**: Está pensada para que los usuarios puedan navegar fácilmente y acceder a la información sin complicaciones.
-   **Filtros dinámicos**: Puedes aplicar filtros rápidamente para buscar lo que necesites de forma más eficiente.
-   **Datos en tiempo real**: Las consultas se realizan en tiempo real a la API, por lo que siempre verás la información más actualizada.
